# Special Wishes

A simple web application for creating and sharing special wishes for any occasion.

## How to Use

1.  **Enter the recipient's name:** Type the name of the person you want to send a wish to in the "Enter recipient's name" field.
2.  **Select an occasion:** Choose an occasion from the dropdown menu. You can choose from Birthday, Anniversary, Congratulations, Thank You, Get Well Soon, or a Custom Message.
3.  **Write a custom message (optional):** If you select "Custom Message," a new field will appear where you can write your own message.
4.  **Generate a wish link:** Click the "Generate Wish Link" button to create a unique link for your wish.
5.  **Copy the link:** Click the "Copy Link" button to copy the link to your clipboard.
6.  **Share the link:** Share the link with the recipient to show them your special wish.

## Features

*   **Customizable wishes:** Create personalized wishes for any occasion.
*   **Beautiful animations:** Enjoy a beautiful fireworks animation when you view a wish.
*   **Responsive design:** The application is fully responsive and works on all devices.
*   **Easy to use:** The interface is simple and easy to use.
